## What changed

- Added a new Activity rail on `/home` for agent and landlord workspace users.
- Activity now highlights recent leads, booking requests, listing approvals, payout milestones, and support escalations.
- Added deep links from each activity row to the relevant workflow (`/host/leads`, `/host/bookings`, `/host/listings`, `/host/earnings`, `/support`).

## Why it matters

- Workspace home now feels live and operational without adding noisy full-panel diagnostics above the fold.
- Hosts and agents can jump directly into urgent actions from one compact stream.
