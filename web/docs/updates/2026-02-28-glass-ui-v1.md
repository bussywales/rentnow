## What changed

- Added a reusable `GlassPill` UI primitive for small floating controls.
- Applied glass styling to Explore action buttons (`save`, `share`, `more`) while keeping existing interactions and testids stable.
- Applied glass styling to the Explore progress indicator.
- Added a subtle glass handle in the Explore `What happens next` micro-sheet header area.

## Why this helps

- Improves perceived polish on mobile without adding heavy visual effects.
- Keeps contrast readable over photography while preserving touch ergonomics and existing layout.
