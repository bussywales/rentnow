## What changed

- Rebuilt `/host/listings` into a dedicated Listings Manager with a portfolio-first default view.
- Added a compact stats strip with portfolio totals and quick status badges (Live, Pending, Draft, Paused).
- Added a persisted `Portfolio | Manage` toggle:
  - `Portfolio`: visual grid with consistent cropped media rhythm.
  - `Manage`: operational table with search, filters, sort, and quick actions.

## Why it matters

- Hosts and agents can scan listing health instantly, then switch into operations without leaving the page.
- The page now behaves like a true dashboard surface instead of a route alias.
