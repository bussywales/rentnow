## What changed

- Added a mobile-first quick-start block at the top of the public home page.
- Added one-tap shortcuts for:
  - Shortlets
  - To rent
  - For sale
  - All homes
- Added a search entry control that routes directly to property browse search.

## Why this helps

- Mobile users can start browsing in one tap without reading long intro sections first.
- The first visible interaction now prioritizes search and category intent.
