## What changed

- Refined the mobile sticky controls on `/shortlets` to use a compact quick-start style shell with tighter spacing and pill controls.
- Reduced visual noise by removing the heavy blur styling and using a calmer, app-like treatment for actions.
- Repositioned the mobile map trigger to a less intrusive corner placement while preserving existing functionality.
- Added regression coverage to guard against oversized sticky-control layouts or duplicate compact bars.

## Why this helps

- Mobile search feels faster and less cluttered while keeping all existing shortlets controls intact.
- Users can still access where, dates, guests, search, filters, and map quickly without the sticky bar dominating the screen.
