## What changed

- Unified filter drawers across `/shortlets` and `/properties` with a shared shell component.
- Standardised filter actions to the same model: `Apply`, `Reset`, and `Clear all`.
- Standardised active-filter chip previews with a shared chip row pattern and clear action.
- Preserved each surface's existing query-param behaviour and filter logic.
- Kept shortlets map/sticky controls and properties browse flow intact while aligning filter UX.
