## What changed

- The host **Getting started** section now auto-hides once all actionable checklist items are complete.
- A small **Checklist complete** chip appears with a **Show again** action so hosts can reopen the checklist anytime.
- Visibility preference is persisted per host on the same browser.

## Why this helps

- Keeps the host home cleaner after setup is done.
- Reduces repeated technical panels while keeping progress context one click away.
