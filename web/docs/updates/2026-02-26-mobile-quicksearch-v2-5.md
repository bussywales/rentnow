## What changed

- Added local-first tracking for featured taps from curated discovery surfaces (home featured strip, shortlets featured rail, properties featured rail, and collections cards).
- Merged featured taps into the quick-search recents feed so first-time and returning users see more useful starting points.
- Added market-aware language in quick search prompts:
  - UK: postcode / flat terms
  - US: ZIP code / apartment terms
  - CA: postal code / apartment terms
  - NG: area / apartment or flat terms
- Added a `Use my last search` CTA in the quick search sheet when local browse/search state exists.
- Improved first-time empty states so users see actionable, market-aware suggestions instead of blank rails.
