## What changed

- Added a reusable host home panel pattern with persisted open/closed state in localStorage (`host:home:panel:<key>:open:v1`).
- Added an `Essentials` strip with quick counts for awaiting approvals, upcoming check-ins, and payout-eligible items.
- Moved secondary sections into cleaner collapsed modules:
  - `Analytics preview`
  - `Demand alerts`
  - `Snapshot`
- Updated Getting Started to auto-expand only when incomplete and stay collapsed by default when complete.

## Why it matters

- Host home is shorter, calmer, and more action-oriented.
- Critical numbers remain visible while technical detail is one click away.
