## What changed

- Updated the Featured strip on `/host?view=all` with inset snap padding so cards no longer start flush to the edge.
- Added trailing spacing to preserve clean snapping while keeping a partial-card peek affordance.
- Kept existing listing actions and card sizing unchanged.

## Why

- Improves scanability with a more intentional, premium carousel rhythm.
- Preserves smooth touch/trackpad swipe without introducing page-level horizontal overflow.
