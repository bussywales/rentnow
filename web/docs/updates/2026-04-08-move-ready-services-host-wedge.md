## What changed

- Added a narrow host/landlord property-prep request flow for end-of-tenancy cleaning, fumigation, and minor repairs.
- Added curated provider admin controls, deterministic request routing, and tokenized provider responses.
- Kept the launch manual and operator-governed instead of turning it into a broad services marketplace.

## Who it affects

- Host/Agent: Can request vetted property-prep help from host surfaces and track matched or unmatched requests.
- Admin: Can create and approve providers, review requests, and route unmatched requests manually.

## Explicit exclusions

- No tenant requester flow
- No public provider browse flow
- No scheduling, payments, payouts, or reviews
- No removals or move-in support categories in this launch
