## What changed

- Added a Featured strip above the host listings mosaic on `/host?view=all`.
- The strip always renders exactly 6 cards and prioritises actively featured listings first.
- Cards use fixed media ratio with `object-cover`, native scroll-snap swipe, and desktop arrow controls.

## Why

- Gives hosts a faster, premium visual scan of top listings before deeper grid browsing.
- Keeps media rhythm stable with no layout jump and no action-behaviour changes.
