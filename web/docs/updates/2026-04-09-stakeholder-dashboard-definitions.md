## What changed

- Added a plain-English stakeholder help page for the traction dashboard at `/help/admin/analytics/stakeholder-dashboard`.
- Linked it from the admin analytics hub and the existing analytics help page so the team can share one definitions link instead of re-explaining metrics repeatedly.

## Why it matters

- Stakeholders now have a single place to understand what core dashboard terms mean, where numbers come from, and which metrics are directional versus operationally exact.
- This reduces misreads without changing the reporting architecture or dashboard design.
