## What changed

- Added a mobile-only featured discovery strip directly under quick start on the public home page.
- Added curated cards with snap scrolling and peek spacing for faster discovery.
- Wired featured cards to existing browse destinations:
  - shortlet cards route to `/shortlets?...`
  - property cards route to `/properties?...`
- Added unit and go-live smoke coverage for featured strip visibility and navigation.

## Why this helps

- Mobile users reach high-intent browse flows in one tap.
- Home feels more visual and actionable without changing existing backend search logic.
