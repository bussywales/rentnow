## What changed

- Added a shared sidebar layout across `/host/*` routes with quick access to Overview, Bookings, Calendar, Listings, and Earnings.
- Sidebar supports collapsed mode and remembers preference locally per browser.
- Host routes now keep navigation context persistent while moving between pages.

## Why it matters

- Hosts can move between core operational pages with less friction and clearer dashboard context.
