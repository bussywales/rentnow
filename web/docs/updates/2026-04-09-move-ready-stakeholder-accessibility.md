## What changed

- Extended the narrow Move & Ready requester wedge so agents can use the same property-prep request flow safely.
- Added visible workspace navigation and agent-help coverage so stakeholders can find the pilot without hidden URLs.
- Kept the scope fixed to cleaning, fumigation, and minor repairs with the same operator-controlled routing model.

## Why it matters

- Stakeholders can now test the wedge as part of the real platform workflow for landlord, host, and agent roles.
- Agent access now works with delegated property context instead of being a superficial role toggle.
- Discoverability is stronger without turning the feature into a broad services marketplace.

## Explicit exclusions

- No tenant requesters
- No removals
- No public provider browsing
- No scheduling, payments, payouts, or reviews
- No provider monetisation or broad marketplace expansion
