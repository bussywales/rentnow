## What changed

- Removed duplicate workspace pill navigation from host and dashboard surfaces.
- Sidebar navigation is now the single primary route-navigation system for workspace pages.
- Host dashboard keeps contextual actions (`Open inbox`) without reintroducing competing route tabs.

## Why this matters

- Reduces navigation confusion from multiple competing nav systems on the same page.
- Keeps host and agent workspace interaction more consistent across routes.
