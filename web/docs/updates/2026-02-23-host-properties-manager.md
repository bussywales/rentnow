## What changed

- Added a dedicated listings manager route at `/host/properties` for host-side roles.
- Added manager controls for status tabs, search, and sort so hosts can quickly find and edit listings.
- Kept `/host/listings?view=all` as the media-first feed while separating editing workflows into the manager view.

## Who it affects

- Host/Agent: clearer distinction between discovery feed and day-to-day listing management.
