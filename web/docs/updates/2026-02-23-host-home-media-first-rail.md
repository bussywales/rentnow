## What changed

- Reworked `/host` top section into a compact workspace hero with direct `New listing` and `Open bookings` actions.
- Added a new `My listings` media rail directly below the hero so imagery appears above the fold before technical panels.
- Kept all existing host actions and data sources intact while moving discovery-first visuals to the top.

## Why it matters

- Hosts now land on a faster, clearer “what to act on next” surface with listing media visible immediately.
- Technical guidance still exists, but no longer blocks the first-screen listing context.
