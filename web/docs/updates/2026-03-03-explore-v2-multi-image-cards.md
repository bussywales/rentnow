## What changed
- Explore V2 card heroes now use swipeable multi-image galleries instead of a single static hero image.
- Added in-card image progress cues for multi-image listings (`1/N` pill and dots) while keeping card layout stable.
- Preserved smooth native vertical scrolling in the feed with no pager-style gesture locking.

## Notes
- This applies to `/explore-v2` only; `/explore` and `/explore-labs` behavior is unchanged.
