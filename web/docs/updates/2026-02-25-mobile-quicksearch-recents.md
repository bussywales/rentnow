## What changed

- Added recent search suggestions beneath the mobile quick-search location input.
- Added tap-to-fill for recent items and a clear action to reset the recent list.
- Added keyboard-first polish: the location field is auto-focused when the sheet opens and Enter submits search.

## Why this helps

- Frequent mobile searches feel faster and more app-like.
- Returning users can re-run common searches without retyping locations.
