## What changed

- Added claim and status lifecycle handling in the admin support inbox (`new`, `in_progress`, `resolved`).
- Added support request assignment controls in both the table rows and the request drawer.
- Added filtering for status and "Assigned to me" to help small teams triage faster.

## Why it matters

- Admins can now process escalations end-to-end without leaving `/admin/support`.
- Clear ownership and state transitions reduce ticket handoff delays during higher volume periods.
