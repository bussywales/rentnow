## What changed

- Added `/explore` as a tenant-focused prototype feed with one listing per viewport using vertical snap scrolling.
- Added horizontal image gallery support within each listing slide plus compact actions for save, share, and opening details.
- Added an Airbnb-lite details bottom sheet with contextual CTA:
  - `Book` for shortlets
  - `Request viewing` for rent/buy listings
- Added a low-risk Explore entry point in mobile Quick Start chips on home.

## Why this helps

- Gives tenants a faster, immersive browsing loop to validate the hybrid TikTok/Instagram/Airbnb discovery direction.
- Keeps interaction lightweight and local-first by reusing existing save/share/view flows and listing assets.
