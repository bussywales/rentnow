## What changed

- Added a tenant-first install experience that appears only after intent, with a 7-day cooldown when dismissed.
- Added iOS Safari guidance for Add to Home Screen when install prompts are unavailable.
- Added an in-app update toast with a refresh action when a newer service worker activates.

## Why this helps

- Install and update behaviour is now easier to understand without interrupting browsing.
- iPhone users get clear setup instructions without unsupported permission prompts.
- Tenants get a predictable path to apply updates without surprise reloads.
