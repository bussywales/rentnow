## What changed

- Added a floating support widget mounted globally across the app.
- The widget opens a compact help panel with search input, quick support actions, and a direct link to `/support`.
- Added keyboard-safe close behavior (`Escape`) and focus return to keep the interaction accessible.

## Who it affects

- Tenant/Host/Agent:
  - Faster access to support entry points from any page.

## Where to find it

- Site-wide floating support button (bottom-right)
- `/support`
