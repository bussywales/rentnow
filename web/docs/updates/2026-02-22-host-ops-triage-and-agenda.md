## What changed

- Host bookings inbox now highlights response SLA risk for awaiting approvals and splits the list into `Urgent` and `Later` sections.
- Hosts can select multiple awaiting requests and run guarded bulk approve/decline actions with progress feedback.
- Host calendar now opens a day agenda drawer/sheet on date click, showing arrivals, departures, stays in progress, and active blocks.
- Agenda items include a quick `Open booking` deep link back to the host inbox.

## Why this matters

- Hosts can act faster on time-sensitive approvals, reducing expiry risk.
- Tenants benefit from quicker host responses and fewer stale pending requests.
