## What changed

- Added a **Brand & Socials** section in admin settings for:
  - Instagram
  - YouTube
  - TikTok
  - Facebook
  - WhatsApp
- Added global rendering of configured social links:
  - Footer: **Follow us** icon row
  - Mobile hamburger drawer: **Connect with us** group
- Links render only when configured; empty settings remain hidden.

## Who it affects

- Admin:
  - Can manage social links centrally from `/admin/settings`.
- Tenant/Host/Agent:
  - Sees social links in footer and menu only when configured by admin.

## Where to find it

- `/admin/settings`
- Global footer
- Mobile hamburger drawer
