## What changed
- Redesigned `/admin/settings` into grouped accordion sections instead of one long stacked page.
- Added a sticky desktop sidebar and mobile section chips for faster navigation between settings areas.
- Added instant "Search settings..." filtering with clear search and desktop expand/collapse controls.

## Notes
- This is a UX-only refactor: setting keys, defaults, save behavior, and APIs are unchanged.
