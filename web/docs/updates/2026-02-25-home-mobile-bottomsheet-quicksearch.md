## What changed

- Replaced the mobile quick-start search page jump with an in-page bottom-sheet flow.
- Added quick intent chips (To rent, For sale, Off-plan, All homes) and location input.
- Added direct fallback action to open shortlets from the same sheet.

## Why this helps

- Search entry is faster and feels native on mobile.
- Users can set intent and location before navigating to results.
