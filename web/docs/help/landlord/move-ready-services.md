## What this pilot is

Move & Ready Services is a narrow property-prep request-routing pilot.

It is for getting a property ready between occupancies, not for booking every service around a move.

## Who can use it

Landlords and hosts can use the current pilot for properties they manage. Agents have a separate agent guide for delegated portfolio use.

## What is currently in scope

Only these categories are in scope:

- end-of-tenancy cleaning,
- fumigation / pest control,
- minor repairs / handyman.

## How requests work

1. Open the property-prep flow from the host workspace or listings manager.
2. Choose one of the three pilot categories.
3. Submit a short request with the property details.
4. PropatyHub ops routes the request to vetted providers that match the category and area.

## How provider matching works now

This pilot is curated. It is not a public provider marketplace.

That means:

- you do not browse providers directly,
- the platform does not auto-book or auto-schedule the job,
- ops may still route some requests manually.

## What to expect if no provider is matched immediately

If there is no immediate provider fit, the request can remain in a manual-routing state while ops checks vetted coverage.

That does not mean your request disappeared. It means the pilot has not found the right provider match yet.

## What this pilot does not do yet

- no tenant request flow,
- no removals or move-in support,
- no public provider browse,
- no scheduling,
- no payments,
- no ratings or reviews.
