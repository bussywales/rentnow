## Listing not showing publicly

- Confirm approved + active status.
- Check expiry and required legal states.
- Ensure listing is not marked demo.
- Verify location/media fields are valid.

## Featured request blocked

- Open eligibility hints in request modal.
- Fix photo minimum or description length gaps.
- Confirm listing ownership and status.
- Retry once after save/refresh.

## Paid but not featured yet

1. Wait for webhook processing window.
2. Check payment status in host panel.
3. Ask admin to run reconcile for your reference if delayed.

## Receipt missing

- Check spam/promotions folder.
- Verify account email is correct.
- Confirm payment is marked succeeded.
- Admin can reconcile receipt-only backlog.

## Escalation bundle

Provide:

- listing ID,
- payment reference,
- request ID (if any),
- timestamp,
- current status screenshot.
