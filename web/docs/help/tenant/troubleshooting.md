## I cannot save a listing

- Confirm you are signed in.
- Re-open the save modal and retry once.
- Check if the listing is still active/visible.
- If persistent, capture listing URL + timestamp for support.

## My shared collection link does not open

- Validate the full `/collections/[shareId]` URL.
- Confirm the collection owner still has sharing enabled.
- If link is old, request a fresh share link.

## Saved search did not create

- Ensure at least one filter or clear search intent exists.
- Retry from `/properties` with specific filters.
- Check plan/role limits if visible in UI.

## Alert digest missing

- Confirm search is active and alerts are enabled.
- Check spam/promotions folder.
- Verify there were actual new matches since last baseline.

## Escalation package

Send support:

- account role and user ID,
- page URL,
- search params or listing ID,
- timestamp,
- screenshot/error text.
