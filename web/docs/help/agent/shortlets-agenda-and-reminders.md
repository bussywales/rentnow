## Agenda workflow

Agents acting for hosts can use `/host/calendar` agenda buckets to focus on:

- urgent pending approvals near check-in
- confirmed arrivals in the next few days

Use booking deep links to move directly into decision workflows.

## Reminder behavior

Reminder sends are automated and idempotent:

- guests receive timing reminders before check-in and checkout
- hosts get a 24-hour heads-up before arrival

## Manual check-in share

For confirmed paid bookings with available check-in data, agents can trigger:

- **Send check-in details now**

This sends guest reminder content without introducing chat.
