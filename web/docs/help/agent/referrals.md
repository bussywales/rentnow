## Who can use referrals

Referral rewards are available to agent workspaces.

If your current workspace is not an agent workspace, you can still see referral-related references in older materials, but the live rewards flow is built around the agent referrals dashboard.

## How referrals are tracked

PropatyHub tracks referrals through your referral code and share link.

Campaign tags can also be added so you can compare which channels are driving clicks, captures, and active referrals.

## When rewards apply

A sign-up alone does not create referral rewards.

Rewards apply only after a qualifying verified paid event is recorded for your referral and the current referral settings allow that reward.

## Where to check status

Use [`/dashboard/referrals`](/dashboard/referrals) to check:

- your referral link and code,
- active referral count,
- milestone progress,
- credits earned and used,
- leaderboard visibility,
- cashout availability for your jurisdiction.

## What is not guaranteed

Referral rewards are not guaranteed just because someone:

- clicked your link,
- created an account,
- or started using the product.

Credits, milestone unlocks, and cashout availability depend on:

- verified qualifying activity,
- current referral settings,
- current jurisdiction policy,
- fraud and abuse controls.

## Important notes

- Cashout rules can vary by country.
- Some jurisdictions can allow credits while still blocking cashout.
- Admin can pause or change referral settings.
