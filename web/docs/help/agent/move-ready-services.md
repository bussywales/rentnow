## What this pilot is for

Agents can use Move & Ready Services to request narrow property-prep help for delegated portfolio properties.

## Current scope

The pilot only covers:

- end-of-tenancy cleaning,
- fumigation / pest control,
- minor repairs / handyman.

## How agents use it

1. Open **Prep Services** from the workspace or portfolio flow.
2. Keep the correct delegated portfolio selected.
3. Submit a short request for one of the three in-scope categories.
4. Track whether the request was matched or still needs manual routing.

## How provider matching works now

The pilot is curated by ops.

You do not choose from a public provider directory. Providers are routed only if they are already vetted for the category and area.

## What if no provider is matched yet?

If no provider matches immediately, the request can remain in manual routing while ops checks coverage.

That means the request still exists, but it has not been matched yet.

## What this pilot does not do yet

- no tenant request flow,
- no public provider browse,
- no scheduling,
- no payments,
- no ratings or reviews.
