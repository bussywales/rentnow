# Agent getting started

<Callout type="info">
This guide covers the quickest path from onboarding to measurable activity.
</Callout>

<Steps>
1. Complete your profile basics and contact details.
2. Publish your first quality listing with media and location.
3. Start handling leads and viewing requests daily.
4. Share your referral link to begin growing Active referrals.
</Steps>

## What success looks like in week 1

- At least one published listing
- Faster lead response time
- Referral link shared with qualified peers
