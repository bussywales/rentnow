# Saved searches & alerts

## Why use saved searches

- Track markets or property types that matter to your clients.
- React quickly when matching listings appear.
- Build a repeatable follow-up loop.

## Setup steps

<Steps>
1. Apply filters for location, type, and price.
2. Save the search with a clear name.
3. Enable alerts and review new matches daily.
</Steps>
