# How to request viewings / respond to leads

## Response cadence that works

<Steps>
1. Acknowledge new enquiry quickly.
2. Confirm preferred date/time windows.
3. Share key listing details before meeting.
4. Follow up after viewing with next steps.
</Steps>

## Before escalating a blocked request

- Confirm listing is still live and available.
- Re-check date and timezone details.
- Capture screenshots and exact error text.
